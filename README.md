# Nirmal & Sumi's Wedding Website

A wedding website for Nirmal & Sumi, celebrating on 6th & 7th September 2026 at Subhasree Hall, Ramnagar, Coimbatore.

Built with [Astro](https://astro.build) and [Tailwind CSS](https://tailwindcss.com).

## Pages

- **Home** — countdown, event details, quick links
- **Our Story**
- **Schedule**
- **Travel & Stay**
- **Registry**
- **RSVP**
- **Guestbook**
- **FAQ**
- **Livestream**

## Development

```bash
npm install
npm run dev      # start local dev server
npm run build    # build for production
npm run preview  # preview production build
```
